# Schutzziele

## Vetraulichkeit

Nur berechtigte Personen können die entsprechenden Daten lesen / schreiben. Anderen wird der Zugriff verweigert.

## Integrität

Die Daten sind konsistent und werden nicht ungewollt oder ohne Meldung verändert werden.

## Verfügbarkeit

Zugriff auf Daten und Systeme ist jederzeit möglich
z.B. ich kann immer auf mein Geld / Kontostand zugreifen
